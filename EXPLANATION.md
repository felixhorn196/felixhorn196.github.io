# HOW TO USE YOUR SITE (Super Easy!)

## Add Content (30 seconds):
1. Visit: https://yourusername.github.io/admin/
2. Click "Login with GitHub" (one-time).
3. Pick collection: Stories, Art, Videos, Theories.
4. "New [Type]" → Fill title, text/images/video embed → "Publish".
5. Live immediately! No coding.

## Examples:
- **Story**: Title "My Bionik Theory", write Markdown.
- **Art**: Title "My Drawing", upload multiple images, description.
- **Video**: Paste YouTube iframe: <iframe src="https://youtube.com/embed/ABC"></iframe>
- **Theory**: LaTeX math with $E=mc^2$, tags like "bionik".

## Edit Repo:
- Only if changing username in admin/config.yml.
- git add . && git commit -m "update" && git push

## Customize:
- Add CSS in _includes/styles.css, push.
- Site auto-builds on GitHub.

Perfect for your bionik work, art, everything! 🚀
