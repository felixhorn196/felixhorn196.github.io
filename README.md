# Felix\'s Portfolio Site

## Quick Deploy:
1. Create GitHub repo `yourusername.github.io` (public).
2. `git remote add origin https://github.com/yourusername/yourusername.github.io.git`
3. `git add . && git commit -m "Deploy site" && git push -u origin main`
4. Edit `admin/config.yml`: replace `yourusername`.
5. Push again. Site live!

## Manage Content:
https://yourusername.github.io/admin/

## Local Test:
npm install
npm run serve
